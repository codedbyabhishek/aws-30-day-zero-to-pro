# AWS 30-Day Zero-to-Pro Checklist

A beginner-friendly 30-day AWS learning roadmap in checklist format.

## Features

- 30 day-by-day tasks from basics to advanced topics
- One-page checklist website
- Progress bar and saved checkmarks with localStorage
- Ready for GitHub Pages

## Run locally

Open `index.html` in your browser.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. Open repository `Settings` -> `Pages`.
3. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main` and folder `/ (root)`
4. Save.
5. Your page will be available at:
   `https://<your-github-username>.github.io/<repo-name>/`
